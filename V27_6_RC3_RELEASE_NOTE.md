# WITH Welfare Mall V27.6 RC3 긴급복구

## 목적
Supabase 주소/키 불일치 및 app_state 테이블 복구 이후, V27.4~V27.6에서 요청된 운영 기능을 안정적으로 재반영합니다.

## 반영 사항
- 복지신청 접수 후 Supabase 즉시 저장 시도 보강
- 복지신청 접수 시 관리자 알림 생성
- 관리자 페이지 상단 운영데이터 새로고침 버튼 추가
- 이벤트 신청 취소 시 신청 기록 삭제
- 복지신청 취소 시 신청 기록 삭제
- 회원 내역과 관리자 현황에서 함께 제거
- 승인/반려 처리 전 신청만 취소 가능
- 회원/관리자 탭에 일반 회원 가입/삭제 목록 추가
- 알림센터 읽음/삭제/전체삭제 추가
- 관리자 알림설정 탭 카카오 발송 대기 이력 삭제/전체삭제 추가
- 비밀번호 초기화 요청 처리완료 버튼 복구
- 기존 기능/자료 유지

## 주의
Cloudflare config.js, Variables and Secrets의 SUPABASE_URL / SUPABASE_ANON_KEY는 운영 Supabase 프로젝트와 동일해야 합니다.
