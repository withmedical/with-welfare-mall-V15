# WITH Welfare Mall V28.0.1 Production

## 변경 사항
1. 관리자 카카오 설정에서 19개 Template ID 직접 수정 및 저장 기능 추가
2. SOLAPI_API_KEY / SOLAPI_API_SECRET 관리자 직접 입력 UI 복구
   - Cloudflare 환경변수가 존재하면 서버 환경변수를 우선 사용
   - 환경변수가 없을 때 관리자 입력값을 보조 방식으로 사용
3. 일반회원 전용 로고 업로드 및 기본 로고 복귀 기능 추가
   - 일반회원 로그인 이후 상단 헤더에 별도 적용
4. 로그인 및 서비스 선택 화면 상단 공통 로고/문구 제거
5. 서비스워커 캐시를 V28.0.1 전용으로 분리
