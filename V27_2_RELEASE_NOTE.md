# WITH Welfare Mall V27.2 Release Note

## 적용 내용
- 첫 화면 글씨 크기 조정
  - “필요한 서비스를 선택하세요.” 약 30% 축소
  - “사계펜션 / 예약”, “직원 / 복지몰” 줄바꿈 적용
  - 카드 제목 크기 약 10% 축소
- 일반고객 회원가입 시 대한민국 기준 성년 여부 확인 추가
  - 만 19세 미만은 가입 불가
- 일반/직원 예약조회에서 예약취소 버튼 표시 개선
- 체크인/체크아웃 버튼 조건 개선
  - 체크인은 체크인 당일 관리자 설정 시간 이후 가능
  - 체크아웃은 체크인 완료 후 가능
  - 체크아웃 당일 설정 시간 이후 미체크아웃 시 알림 생성 구조 추가
- 관리자 숙소관리 > 숙소 기본 설정에 체크인/체크아웃 시간 추가
  - 기본/최대 인원과 함께 관리
  - 예약 화면 및 체크인/체크아웃 버튼 조건에 연동

## 유지 사항
- 기존 직원 복지몰 기능 유지
- 기존 예약/복지/이벤트/공지 데이터 유지
- 기존 관리자 기능 유지
