# WITH Welfare Mall V27 Release Note

## Theme
사계펜션 예약 서비스 오픈 / 일반고객 예약入口 추가

## 주요 반영
- 첫 화면을 `사계펜션 예약` / `직원 복지몰 입장`으로 분리
- 일반고객 로그인 및 회원가입 추가
- 일반고객 회원가입 최소 정보 수집: 이름, 생년월일, 휴대폰, 비밀번호
- 개인정보 수집·이용 동의, 이용 완료 후 7일 이내 폐기 동의, 환불규정 동의, 만 14세 이상 확인 추가
- 일반고객 로그인 후 숙박 예약 관련 기능만 표시
- 직원전용은 기존 복지몰 구조 유지
- 관리자 좌측 메뉴에 `알림설정` 탭 추가
- 카카오 알림 연동 준비 항목을 알림설정 탭에서 관리

## 데이터 보호
- 기존 직원/관리자/예약/복지/이벤트/공지 데이터 유지
- 신규 일반고객 데이터는 `customers` 구조로 추가
- 기존 직원용 기능 삭제 또는 축소 없음
