# V27.6 Release Note

## Critical Fix
- iPhone Safari에서 신청 후 관리자 화면에 보이지 않거나 로그아웃 후 사라지는 문제를 줄이기 위해 중요 신청 데이터 저장 시 Supabase 즉시 저장을 기다리도록 보강했습니다.
- 숙소예약, 복지신청, 이벤트 신청/취소는 저장 실패 시 안내 메시지를 표시합니다.

## Admin
- 회원/관리자 탭에 일반회원 가입/삭제 목록을 추가했습니다.
- 알림센터에 읽음, 삭제, 전체 삭제 기능을 추가했습니다.
- 알림설정 탭의 카카오 발송 대기 이력에 삭제, 전체삭제 기능을 추가했습니다.

## PWA
- 휴대폰 홈 화면에 앱처럼 추가할 수 있도록 manifest, service worker, 앱 아이콘을 추가했습니다.
- iPhone: Safari 공유 버튼 > 홈 화면에 추가
- Android: Chrome 메뉴 > 홈 화면에 추가

## 유지사항
- 기존 직원 복지몰 기능과 기존 데이터 구조는 유지했습니다.
