# V23 변경사항

- 예약 달력 날짜 하루 밀림 보정
  - 날짜 계산을 로컬 날짜 기준으로 통일
  - Google ICS의 DTEND 종료일 처리 유지
- 예약 출처별 색상 표시 개선
  - 복지몰, Google, Airbnb, 수기예약, 외부, 예약차단 구분
  - 관리자는 예약자/출처를 달력에서 확인 가능
  - 일반 회원은 본인 예약만 식별하고 타 예약은 예약불가로 표시
- 숙소 기본/최대 인원 설정 추가
  - 관리자 > 숙소 관리에서 숙소별 기본인원/최대인원 수정 가능
  - 직원 예약 화면과 예약 인원 제한에 즉시 반영
- 카카오 알림 연동 준비 구조 추가
  - 실제 발송은 보류
  - 웹 알림, 카카오 발송대기, 예약 타임라인 데이터 구조 준비
- 기존 기능과 기존 localStorage/Supabase 데이터 유지 방식 적용
