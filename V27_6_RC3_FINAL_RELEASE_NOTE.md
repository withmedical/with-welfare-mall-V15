1. 직원/일반 신청 저장 시 성공 안내 전에 localStorage 저장과 Supabase 저장 결과를 확인하도록 변경
2. iPhone/Safari 저장 실패 시 성공 메시지를 띄우지 않고 오류 안내
3. 비밀번호 초기화 요청을 운영 데이터 저장 경로(saveCriticalData)로 처리
4. 관리자 비밀번호 초기화 `처리완료` 버튼 누락 함수 복구
5. 관리자 할인관리 탭 복구
6. 직원 할인 페이지가 홈으로 강제 이동되던 문제 수정
7. 할인 항목의 신청 가능 여부/제한 인원 값이 마이그레이션 중 사라지지 않도록 보호
8. Supabase 저장 실패 메시지에서 정의되지 않은 err 참조 오류 제거
